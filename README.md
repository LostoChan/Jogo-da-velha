# Jogo-da-velha
Jogo da velha que criei como um projeto para a faculdade.

O jogo conta com modo Jogador vs Jogador e Jogador vs Máquina.

Lógica da Máquina (IA)

A máquina foi programada com uma lógica simples porém eficiente:

Tenta jogar no centro se estiver livre.

Verifica se pode vencer na próxima jogada e joga na posição vencedora.

Verifica se o jogador pode vencer e bloqueia a jogada.

Caso nenhuma condição acima seja possível, faz uma jogada aleatória válida.


Funcionalidades

Sistema de tabuleiro 3x3.

Verificação de vitória (linhas, colunas e diagonais).

Detecção de empate (deu velha).

Verificação de posições válidas (sem aceitar jogada repetida).

Pergunta se quer jogar novamente ao final da partida.

Interface totalmente no console.


O projeto foi desenvolvido sem usar POO, conforme exigência da disciplina — utilizando apenas conceitos de:
métodos,
arrays,
condicionais,
loops,
modularização.
